# SnapCity Mobile Prototype

A clickable React + Tailwind CSS prototype for the SnapCity civic reporting flow:

Camera -> AI Scanning -> AI Case Ticket -> Reward -> Case Dashboard.

## Run

Serve the folder with the included zero-dependency Node server, then open the local URL:

```powershell
node server.mjs
```

Then visit `http://localhost:5173`.

You can also run `npm.cmd start` on Windows if PowerShell blocks `npm`.

The prototype uses CDN-hosted React, Babel, Tailwind, and Inter for a zero-install local preview.
